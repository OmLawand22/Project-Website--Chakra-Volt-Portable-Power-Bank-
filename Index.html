Key Features

Explore the core components that make CHAKRA-VOLT powerful, portable, and smart.

🔋 Real-Time Voltage Monitoring

Continuously tracks the EV battery voltage with high accuracy using ADC and voltage divider.

🌡️ Temperature Sensing

Monitors ambient temperature with a DHT11 sensor to ensure safe operation of the system.

📲 IoT Dashboard (Blynk)

Live data is pushed to a mobile dashboard using Blynk IoT platform, accessible via QR code.

⚡ Auto Cut-Off

Built-in logic automatically cuts power when voltage crosses safety limits, protecting the battery.

📦 Portable Design

Compact layout fits easily into storage compartments of electric bikes or scooters for emergencies.

🛠️ Scalable Architecture

Can be upgraded to support higher voltage (24V/48V) systems with minimal changes in hardware.

🔩 Hardware Design

Our system uses ESP8266 microcontroller with DHT11 sensor and voltage divider circuitry. It’s connected to a DC-DC booster and battery module, and is compact enough to be carried for field deployment.

💻 Software Integration

CHAKRA-VOLT uses Arduino code for sensor interfacing and data transmission. Blynk IoT platform displays real-time voltage, temperature, and charging status through mobile dashboard using Wi-Fi.

🚀 Future Scope

We aim to implement solar charging, LiFePO4 batteries, waterproof casing, and higher voltage capacities to make CHAKRA-VOLT usable for a wider range of electric vehicles and rural rescue kits.

Technologies Used

Here's a look at the tools, platforms, and components used to build CHAKRA-VOLT.

ESP8266 Microcontroller

Wi-Fi-enabled microcontroller for sensor interfacing and IoT integration.

DHT11 Sensor

Measures ambient temperature and humidity for safety monitoring.

Voltage Divider Circuit

Steps down battery voltage to a measurable range for the ESP8266 ADC.

Blynk IoT Platform

Cloud platform used to display sensor data in real time via smartphone.

LiquidCrystal I2C LCD

Displays voltage, temperature, battery level and charging status locally.

Arduino IDE

Open-source development platform for writing and uploading code to ESP8266.
